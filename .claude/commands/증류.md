증류를 수행하라.

1. `python3 tools/distill.py --dry-run` 으로 이번에 처리할 새 파일 목록을 확인한다.
   - `.state.json`은 엔진 실행 기준이다. 대화형으로 증류하는 경우, 끝난 뒤 어떤 파일을 처리했는지 보고에 명시하라.
2. `tools/distill_prompt.md` 를 읽고 **그 지시서대로** 수행한다 (주장 추출 → 4분류 대조 → STATUS 전체 재작성 + 질문 병합 추가).
3. 큰 HTML은 파이썬으로 스타일·스크립트 제거 후 텍스트만 추출해 분석한다.
4. **documents/facts/ 는 절대 직접 수정하지 않는다** — 병입은 별도 명령.
5. `documents/업데이트로그.md` 맨 위에 1줄 추가(`🌀 반영`).
6. 끝나면 `python3 -m mkdocs build`로 검증하고, 보고한다:
   `새 주장 N건 · 충돌 N건 · 중복 소멸 N건 · 추천 안건 순위`

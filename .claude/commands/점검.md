위키 사이트를 감사하라. 발견만 보고하고, 수정은 운영자 확인 후 진행한다.

## 1단계 · 기계 검사 (몇 초)

`python3 tools/lint.py` — nav 커버리지·깨진 링크·심볼릭·NFC 충돌·결정번호 연속성·안건코드 정합·운영용어·낡은 참조를 한 번에 검사한다.
이어서 `python3 -m mkdocs build --strict` (Material 팀 공지 경고는 무시).

> lint가 오탐을 내면(정당한 이력 언급 등) 코드를 우회하지 말고 `tools/lint.py`의 `CONFIG` allow 목록에 근거와 함께 추가하라.

## 2단계 · 판단 검사 (LLM — lint가 못 잡는 것)

`tools/distill_prompt.md` 중복·언어 규칙 기준:

- **한 사실 한 곳**: 열린 안건·확정 목록이 질문.md·결정로그.md 밖에 복붙돼 있는지
- **문체**: 1인칭("나", "내 ") 잔존, 벽글 인용문, admonition 없는 벽글 머리
- **정합**: 페이지 간 내용 모순 (예: 파이프라인 단계 수, 테이블 수, 담당자)
- **낡음**: 사실은 바뀌었는데 문장이 옛 상태를 설명하는 곳

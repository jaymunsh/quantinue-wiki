---
description: 검증 후 GitHub Pages 배포 (jaymunsh.github.io/quantinue-wiki)
---

위키를 배포하라.

1. 미커밋 변경이 있으면 먼저 보고하고 커밋 여부를 확인받는다.
2. `python3 -m mkdocs build` — 에러·링크 경고 0 확인. 경고가 있으면 **배포 중단**하고 보고.
3. `python3 -m mkdocs gh-deploy --force` 실행.
4. 배포 후 라이브 확인: `curl -s -o /dev/null -w "%{http_code}" https://jaymunsh.github.io/quantinue-wiki/` 가 200인지 (반영에 1~2분 걸릴 수 있음).
5. 배포된 커밋 해시와 함께 결과 보고.

## 3. 지금 (P0)

### P0-1. 첫 CI 배포 검증 🚀
`deploy.yml`은 작성만 됐고 **한 번도 실행된 적 없다.** 첫 push 전 확인:
1. GitHub repo → Settings → Actions → General → **Workflow permissions = "Read and write"** (안 켜면 `gh-pages` push가 권한 에러)
2. push 후 Actions 탭에서 `deploy-wiki` 성공 + 라이브 200 확인
3. 성공하면 이 커밋에 **`git tag gate1`** 소급(§4 D와 합침)

- 공수: 설정 토글 + 관찰 (~10분) · 시점: **다음 배포 때 바로**

### P0-2. `facts/화면.md` 채우기 🖥️
현재 **5줄짜리 빈 stub**인데 멘토 안건 **C3(화면·사용자 흐름 산출물)** 의 대상. 발표 산출물에도 필요.
- 최소: 화면 목록 + 사용자 플로우 1개(mermaid) 골격. 확정 아닌 건 `[미정]`
- 공수: ~30분 · 시점: 이번 주 (C3 논의와 함께)
